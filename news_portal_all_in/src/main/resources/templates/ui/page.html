About Us

Hello.

We present to your attention version 1.0 of the news portal project. This project is the result of a team development of 4+ people. We are a novice backend development team and therefore we have not fully implemented the interesting elements of the frontend. Much of what we planned we did not have time to implement. The main functionality is implemented and presented to you for consideration.

The project implemented:
1. View of news.
2. Search news.
3. Working with the user (log in, register, change the user).
4. Rich Text Editor for working with news(create, edit, delete).
5. Viewing user statistics.
6. Like news, comments.
7. Write a comment or reply to a comment.
